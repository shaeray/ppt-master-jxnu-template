## I. Template Overview

**Template ID**: 江西师范大学

**Use Cases**: 毕业答辩、开题报告、中期汇报、科研汇报、课程教学、学术交流.

**Visual Personality**: 正式、学术、清爽。画面应像一份大学官方母板的延展版本：稳定的留白、克制的校色、清晰的页眉页脚，以及可被内容快速接管的主体区域。封面和章节页保留校园影像带来的庄重开场；正文页减少装饰，把注意力交给标题、图表、论证结构和研究材料。

**Replication Mode**: fidelity. 原始母板有封面、目录、章节、标准正文、有底部校训正文、无底部校训正文、结束页等明显差异，模板保留多种正文变体供不同学术内容切换。

## II. Color Scheme

**Primary**: `#A13838`，作为标题条、章节编号、页码、强调线和关键节点色。

**Secondary Rose**: `#C16E6E`，作为目录渐变、次级编号和轻量强调的辅色。

**Soft Rose**: `#E9CAD7`，用于淡色背景块、图表底纹和目录编号底色。

**Academic Ink**: `#262626`，用于正文主文本。

**Quiet Gray**: `#7A7A7A`，用于辅助说明、来源和页脚信息。

**Campus White**: `#FFFFFF`，用于深色影像上的标题、校名和结尾页信息。

## III. Typography

**Primary Chinese**: Microsoft YaHei. 原母板正文与说明页使用微软雅黑体系，模板保持这一官方气质。

**Body Chinese**: Microsoft YaHei Light where available, falling back to Microsoft YaHei.

**Latin Fallback**: Arial. 适用于英文摘要、图表标签和跨平台 PowerPoint 打开场景。

**Type Feel**: 标题应偏稳重，字号较大但不拥挤；正文区域保持中等字号和足够行距，适合科研报告中的长标题、术语和多级论证。

## IV. Signature Design Elements

**Campus Image Opening**: 封面和结束页使用校园航拍影像叠加深色罩层，呼应官方母板的开场方式。背景取景应让图书馆立面上的校徽落在页面视觉中轴，与前景白色校徽和居中文案保持纵向一致。

**Half-Image Chapter Divider**: 章节页采用上半幅校园图与下半幅白底标题区，保留原母板中横向分割线的秩序感。章节编号与章节标题之间保持明显横向间距，避免编号贴近标题。

**Top-Right University Wordmark**: 正文页右上角固定使用母板导出的江西师范大学横向校名标识，形成连续的学校身份。

**Motto Footer**: 标准正文页底部使用母板导出的“静思笃行 持中秉正”校训图像，作为低调但明确的品牌记忆点。

**White Motto Mark**: 封面、目录页和结束页在深色背景上使用纯白校训图像。章节页不放左上品牌标识，把视觉重心交给章节编号和标题。

**Rose Rule System**: 标题条、页码、目录编号和时间轴节点统一使用主色与辅色构建页面骨架，避免过度装饰。

## VII. Original Preview Compatibility

**Deployment Rule**: 为兼容原版 ppt-master live preview，最终生成到 `svg_output/` 的页面不得引用 `templates/`、`../templates/` 或模板 SVG 的同目录裸文件名。原版预览服务只稳定映射项目级 `images/` 与 `assets/`，因此模板图片资产必须先部署到 `<project>/images/`，再由最终页面引用。
